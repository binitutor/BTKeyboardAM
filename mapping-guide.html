Keyboard Mapping Guide

How It Works

The BT Amharic Keyboard supports physical keyboard input that automatically translates English phonetic keys to Amharic characters. Simply type using your English keyboard, and the application will convert your input to Amharic in real-time.

Example: Type h + a to get ሀ

Pro Tips
  • Quick Typing: Once you learn the patterns, you can type very quickly using your physical keyboard
  • Single Letters: Press just the consonant (e.g., m) to get the schwa form (ም)
  • Vowel Combinations: Different vowel combinations create different character forms
  • Mix Input Methods: You can combine virtual keyboard clicks with physical keyboard typing
  • Buffer Reset: The input buffer automatically clears after a match to prepare for the next character
Complete Reference

The complete keyboard mapping is stored in kbdmapping.json. This file contains all supported phonetic combinations and their corresponding Amharic characters.

Note: The mapping file loads automatically when you open the keyboard. If you modify the JSON file, refresh your browser to reload the mappings.
